# 🧶 Mini Crochetter – Handmade E-Commerce Website

A fully functional frontend e-commerce website for handmade crochet products.  
Built using HTML, CSS, and JavaScript with cart system and WhatsApp checkout integration.

## ✨ Features

- 🛍 Product listing with categories
- 🖼 Product images with popup preview
- 🛒 Add to cart system (localStorage based)
- ➕ / ➖ quantity control
- 🧾 Order summary page
- 📲 WhatsApp checkout integration
- 🔍 Category filtering
- 📱 Responsive layout (basic mobile support)

## ⚙️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API
- WhatsApp API (wa.me)

## 🚀 How to Run

1. Clone the repository
2. Open `index.html` in browser
3. Or use Live Server (VS Code)
